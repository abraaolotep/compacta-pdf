# Compacta PDF — LOTEP

Ferramenta da LOTEP (Loteria do Estado da Paraíba) que reduz PDFs para no máximo **9,9 MB** e, quando a compactação não é suficiente, divide o arquivo em volumes.

**Acesse:** https://abraaolotep.github.io/compacta-pdf/

- Processamento 100% no navegador — nenhum arquivo é enviado a servidor.
- Compactação em níveis de qualidade decrescentes (150 DPI/80% até 96 DPI/35%).
- Divisão em volumes sem perda de qualidade quando necessário.
- Limite ajustável (padrão 9,9 MB, seguro para PJe, SEI e e-mail).
- Funciona offline: baixe o `index.html` e abra com duplo clique.
